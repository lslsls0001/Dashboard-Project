step 2: open a terminal with the manage.py, and in the terminal, run python3 manage.py makemigrations
step 3: run python3 manage.py migrate
step 4: run python3 manage.py runserver
step 5: run "npm run dev" on api_client terminal
step 6: in the browser, enter the url: http://127.0.0.1:8000/api_client/login
